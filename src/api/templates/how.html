Viberary 

# What is it? Viberary is a semantic search engine that will find books based on mood rather than traditional keywords like genre or author, although it can do that, too! What's a vibe? Hard to define, but anything like a mood, a feeling, a color, a phrase, much like Netflix's categories for movies. A traditional search for a book might be for "scifi", a vibe search might be, "funny sci-fi in space with feeling" Traditional search engines work by doing lexical matching and looking up words against an inverted index that maps each word and the document each word is located in.
